# srsj

Simple Rtmp Server-Java

基于flazr的java版rtmp服务器，取名srsj是蹭一下srs的热度

rtmp推流：暂时只支持简单握手，仅使用obs测试通过

flv拉流：仅支持HTTP-FLV的播放，使用VLC和flv.js测试播放通过
